## Usage

-   Clone it to your PC
-   Edit the `monkey.config.js` **NEED NOT TO EDIT THE `monkey.dev.config.js`**
-   Run `test` script in the `package.json`
-   Enjoy

## Features

-   An awesome way to add style `GM_addStyle(require(filename))`
-   Test without `Ctrl+V`
-   Build a public version easily
